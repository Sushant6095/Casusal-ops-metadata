## How to capture (macOS)

1. Hide the bookmark bar: `⌘⇧B`
2. Browser zoom to 110% on each route
3. Capture full window: `⌘⇧4` then **Space**, click the window
4. Crop the OS chrome out in Preview if needed
5. Save as `<name>.png` here

Recommended dimensions: ~1920×1080 or any 16:9. PNG keeps the gradient
mesh and glow effects crisp.

## Why these four?

They map 1:1 to the four product screens the README references in the
**The product** section. Adding a 5th screenshot? Reference it
explicitly in the README — judges often skim, so every image needs a
caption that explains *why it matters*, not just what it is.
